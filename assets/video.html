1920x1080
Dreamcast creates a 720p EDTV image, but is only using the inner 640 pixels.

The pixel ratio is 1:1, if displayed via VGA adapter.
720x480 1:1
Dreamcast image is pixel doubled to 1280x960 pixels.

The pixel ratio remains 1:1.

As most HDTVs do not support this, you have to either use a Framemeister or implement it on the FPGA.
1440x960 1:1
Dreamcast image is scaled to full size 4:3 by HDTV.

The pixel ratio will be 8:9, so the image will be squashed.
1440x1080 4:3
Dreamcast image is scaled to full size 16:9

The pixel ratio will be 32:27, so the image will be stretched.
1920x1080 16:9
